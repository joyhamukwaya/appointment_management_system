# Python Appointment Management System

## Overview
The Appointment Management System is a console-based application built using Python. 
It allows users to create and manage appointment records in a structured format.

## Features
- Add new appointments
- View existing appointments
- Organized data handling
- Clean and readable code structure

## How It Works
The system allows users to input appointment details through the console interface. 
Appointments are stored and displayed based on user requests.

## Technologies Used
- Python
- Basic data structures
- Console-based interface

## Concepts Demonstrated
- Functions
- Loops and conditionals
- User input validation
- Structured program flow
